# Advanced Linked List Problems

These problems improve understanding of algorithms and pointer techniques.

---

# 1. Reverse a Linked List

## Status
✅ Completed

## Description
Reverse the entire linked list in-place.

## Why This Problem Matters
- One of the most important linked list interview problems
- Builds strong pointer manipulation skills

## Approach
Three-pointer iterative method:
- Previous pointer
- Current pointer
- Next pointer

---

# 2. Detect Cycle

## Status
⏳ To Implement

## Description
Check whether the linked list contains a cycle (loop).

## Why This Problem Matters
- Prevents infinite traversal
- Useful in memory management and debugging

## Approach
Floyd’s Cycle Detection Algorithm (Tortoise and Hare)

---

# 3. Find Middle Element

## Status
⏳ To Implement

## Description
Find the middle node of the linked list.

## Why This Problem Matters
- Common interview problem
- Improves two-pointer technique

## Approach
Fast pointer moves two steps  
Slow pointer moves one step

---

# 4. Merge Two Sorted Lists

## Status
⏳ To Implement

## Description
Merge two sorted linked lists into one sorted list.

## Why This Problem Matters
- Builds comparison and merging logic
- Important for merge sort understanding

## Approach
Iterative merging using a dummy node.
