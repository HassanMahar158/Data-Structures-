## Advanced Linked List Problems (This Week's Focus)

### 1. Reverse a Linked List

* **Description**: Reverse the entire linked list in-place
* **Why**: Core interview problem, tests pointer manipulation
* **Approach**: Iterative three-pointer technique
* **Status**: ✅ Completed

### 2. Detect Cycle

* **Description**: Check if linked list has a cycle (loop)
* **Why**: Useful real-world scenario (memory leaks, infinite loops)
* **Approach**: Floyd's Cycle Detection Algorithm (Tortoise and Hare)
* **Status**: ⏳ To implement

### 3. Find Middle Element

* **Description**: Find the middle node of the linked list
* **Why**: Two-pointer technique practice
* **Approach**: Fast and slow pointer method
* **Status**: ✅ Completed

### 4. Merge Two Sorted Lists

* **Description**: Merge two sorted linked lists into one sorted list
* **Why**: Combines multiple operations, good for interviews
* **Approach**: Iterative merging with dummy node
* **Status**: ⏳ To implement

### 5. Remove Duplicates

* **Description**: Remove duplicate values from unsorted linked list
* **Why**: Practical problem, hash table usage
* **Approach**: Use hash set to track seen values
* **Status**: ✅ Completed

## Advanced Linked List Types

### 6. Doubly Linked List

* **Description**: Implement bidirectional traversal
* **Why**: More flexible than singly linked list
* **Features**: Previous pointers, easier deletion
* **Status**: ⏳ To implement

### 7. Circular Linked List

* **Description**: Last node points back to first node
* **Why**: Different termination condition, useful for round-robin
* **Operations**: Insert, delete with circular logic
* **Status**: ⏳ To implement
